(PT/BR)# 🧮 Weight Conversion System (Java Console Application)

Este projeto é uma aplicação de console desenvolvida em Java com o objetivo de praticar lógica de programação, entrada de dados e estruturas condicionais.

## 🚀 Objetivo

Criar um sistema simples e interativo que permite converter unidades de peso entre libras (lbs) e quilogramas (kg), simulando uma ferramenta prática de conversão.

## ⚙️ Funcionalidades

- Menu interativo no terminal
- Conversão de:
  - Libras → Quilogramas
  - Quilogramas → Libras
- Validação de entrada do usuário
- Exibição imediata do resultado

## 🧮 Fórmulas aplicadas

- 1 lb = 0.453592 kg  
- 1 kg = 2.20462 lbs  

## 🧠 Conceitos praticados

- Estrutura condicional (`if / else`)
- Entrada de dados com `Scanner`
- Manipulação de variáveis (`double` e `int`)
- Boas práticas básicas de fluxo de programa

## 🛠️ Tecnologias

- Java
- Terminal / Console

## 📌 Resultado

O usuário interage com o sistema via terminal, escolhe a conversão desejada e recebe o resultado de forma imediata.

## 👨‍💻 Autor

Projeto desenvolvido para prática de lógica de programação em Java.

(EN)# 🧮 Weight Conversion Program (Java)

This is a simple Java console application that converts weight units between **pounds (lbs)** and **kilograms (kg)**.

## 📌 Features

The user can choose between:

1. Convert pounds (lbs) to kilograms (kg)  
2. Convert kilograms (kg) to pounds (lbs)

After selecting an option, the program asks for the value and displays the converted result.

## ⚙️ Conversion formulas

- 1 lb = 0.453592 kg  
- 1 kg = 2.20462 lbs  

## 🧠 How it works

- User input is handled using `Scanner`
- A simple menu is displayed in the console
- Conditional statements (`if / else if / else`) control the program flow
- The result is calculated based on the selected option
- Invalid inputs are handled with an error message

## 🛠️ Technologies used

- Java
- Scanner (console input)
